## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter
repository.

## Installation

``` bash
$ npm install
```

## Running the app

First, deploy data base with Docker:

``` bash
$ docker-compose up -d
```

Then, run app:

``` bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
